### class
```
동일한 속성과 행위를 수행하는 객체의 집합
객체 : 필드, 메소드
```

종류는 public, private, protected, default가 존재한다.

### static
```
static - class method
no static - instance method
```

static은 클래스에 고정된 멤버로서 클래스 로더가 클래스를 로딩하면 메모리에 할당 시켜준다.
